# ha_qqmail
在HA里使用的QQ邮箱通知

# 使用方式

```yaml

# 配置后生成【ha_qqmail.notify】服务
ha_qqmail:
  qq: QQ号码
  code: QQ邮箱授权码

```

> 调用服务格式
```yaml
title: 标题
message: 消息内容
data:
  url: https://github.com/shaonianzhentan/ha_qqmail
  image: https://vuejs.org/images/logo.png
  actions:
    - action: close_light
      title: 关灯
    - action: close_all
      title: 关掉全部灯
```

> 获取点击事件(参考链接：https://www.home-assistant.io/integrations/html5/)
```yaml
- alias: 这是一个H5的通知提示
  trigger:
    platform: event
    event_type: html5_notification.clicked
    event_data:
      action: close_light
```
